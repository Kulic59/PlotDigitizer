Программа для оцифровки точек на растровом изображении

Основной структурой для хранения выделенных точек является класс TDigPoitnsList,
потомок TObjectList, который хранит коллекция объектов TDigPoint - координаты точек
в растровом виде (это нужно для перерисовки изображения с точками).
Класс TDigPointsList содержит также координаты "осей оцифровки" необходимые для
учета поправок на смещение и поворот осей, а также для оцифровки в миллиметрах.

Преобразование точек происходит при выводе текущего положения курсора и при
сохранении в буфер обмена. Для этого служат свойства X[i] и Y[i].
